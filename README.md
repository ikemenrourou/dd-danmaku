# emby-danmaku

基于 [l429609201/dd-danmaku](https://github.com/l429609201/dd-danmaku) 的个人魔改版本。

## ✨ 新增功能

### Bangumi 角色介绍
集成了 Bangumi 的角色信息展示功能，在播放页面可以查看角色介绍、声优信息等。

![Bangumi角色介绍功能](https://files.catbox.moe/vx1k5k.png)

## 📦 使用方法

下载 `ede.js` 文件，注入到 Emby 客户端即可。

详细安装方法请参考原项目：[l429609201/dd-danmaku](https://github.com/l429609201/dd-danmaku)
